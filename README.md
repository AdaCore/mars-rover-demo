# Mars Rover Demo

A Mars Rover implemented and formally verified in SPARK, along with a
corresponding [simulator](simulator/) implemented in Rust.

If you came here from
[our blog post](https://www.adacore.com/blog/mars-rover-gnc) about our work on
guidance, navigation and control for the Mars Rover, you need to switch to the
`mars-rover-gnc` branch.

If you're here for something you want to run on the Mars Rover hardware, this
branch is for you!
